# NRSTUDIO

Strona główna dla NRSTUDIO — studia budującego strony internetowe dla trenerów personalnych.

Statyczna, jednoplikowa strona (`index.html`, HTML + CSS + vanilla JS, bez zależności i procesu builda).

## Podgląd lokalny

Otwórz `index.html` w przeglądarce albo uruchom lokalny serwer, np.:

```
npx serve .
```

## Uwaga o kalendarzu

Sekcja Kontakt osadza widżet Calendly (`calendly-inline-widget`). Podmień `data-url` w `index.html` na docelowy link Calendly/Cal.com przed wdrożeniem produkcyjnym.
